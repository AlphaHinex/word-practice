使用 words.json 文件中的数据，制作一个英文单词练习 web 应用，只使用基本的 html/js/css 即可，代码集中在单个文件中。

开始练习前，需从如下选项中进行选择：
1. 练习方式：英译汉 / 汉译英
2. 练习内容：全部 / 常错

开始练习后，具体要求如下：
1. 选择英译汉时，界面显示如下字段内容：english, phonetic, tag
2. 选择汉译英时，界面显示如下字段内容：chinese, tag
3. 选择练习常错内容时，界面增加 right 和 wrong 字段值的显示，范围为 wrong 大于 right 的对象
4. 选择练习全部内容时，范围是全部 json 对象
5. 练习时，json 对象随机顺序出现，但不要重复出现，可切换上一个或下一个，直至所有练习内容都练习一遍
6. 当单词是第一个时，不显示上一个按钮；当单词是最后一个时，不显示下一个按钮
7. 每个单词有一对 答对了/答错了 按钮，用来增加 right 和 wrong 的值
8. 当所有内容都练习一遍之后，可对结果进行持久化，主要更新 words.json 文件中每个 json 对象的 right 和 wrong 字段值，供下次练习时的常错选项使用
9. 持久化功能如受限，可单纯的将更新后的 json 集合打印出来，手动进行更新
